# file-explorer

## Setup
- Make an .env file in the root of the project or just copy the .env.example file and rename it to .env
- Also copy the .env.example file in the apps/api folder and rename it to .env

install dependencies:

```bash
bun install
```

Run Migration:

```bash
bun run migration
```

## Running App

```bash
bun run dev
```
